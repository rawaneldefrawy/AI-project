# 🧩 Sudoku Game (Python + Pygame)

A simple and interactive Sudoku game built using Python and Pygame.
The game allows users to play Sudoku, check their solution, or solve it automatically.

---

## 🎮 Features

* 🟦 Interactive Sudoku grid
* 🧠 Backtracking algorithm to solve Sudoku
* ✍️ User input (type numbers manually)
* ✅ Check button to validate solution
* ⚡ Solve button for automatic solving
* 🔄 Generate new random puzzle
* 🎨 Simple GUI with highlighted original cells

---

## 🛠️ Technologies Used

* Python 🐍
* Pygame 🎮

---

## 🚀 How to Run

1. Make sure Python is installed
2. Install pygame:
    pip install pygame
3. Run the game:
    python sudoku.py

## 🧠 How It Works

* The game uses a **Backtracking algorithm** to solve Sudoku
* A complete grid is generated, then numbers are removed randomly to create a puzzle
* The player fills the empty cells and can validate the solution
